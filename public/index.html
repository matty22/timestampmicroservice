Timestamp Microservice

This timestamp microservice takes a URL with a date parameter and returns a JSON string with the unix and natural date formats. To use this microservice, enter a URL that contains a date parameter into your browser URL bar and hit enter.

User stories:

  1. I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it returns null for those properties.

Enter a URL like one of these into your browser URL bar:

https://matty22timestampmicroservice.herokuapp.com/December%2015,%202015
https://matty22timestampmicroservice.herokuapp.com/1450137600

Example output:

{"readable":"December 15, 2015","unixtime":"1450166400"}
